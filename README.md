Lenovo A7000-a - Device Tree Configuration (aio_row)
==============

This branch is for building 7.x based ROMs.

Basic   | Spec Sheet

CPU     | 1.5GHz 64-bit Octa-Core MT6752
GPU     | Mali-T760MP2
Memory  | 2GB RAM
